# Conditionals
source: https://www.w3schools.com/python/python_conditions.asp


## Podstawowe warunki

```
Equals: a == b
Not Equals: a != b
Less than: a < b
Less than or equal to: a <= b
Greater than: a > b
Greater than or equal to: a >= b
```


## Warunki if

W pythonie możemy stawiać warunki za pomocą *if*

```
a = 33
b = 200
if b > a:
  print("b is greater than a")
```
